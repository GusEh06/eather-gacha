# Aether Gacha

Leer PRD.md y DESIGN.md antes de cualquier tarea.

## Reglas absolutas
- Moneda única: Shards. Solo se obtienen via Stripe. Onboarding: 320 Shards gratis.
- Mercado (Bazaar) acepta solo Shards, nunca dinero real.
- Dust y Nebula NO se pueden listar en el Bazaar.
- Transacciones del Bazaar son atómicas en MongoDB.
- El 5% de tributo del Bazaar se destruye, no se redistribuye.
- Tipografía: Cinzel (display) + Rajdhani (UI). Nunca Inter, Roboto ni System UI.
- Usuarios se llaman "Aether Binder [username]" — siempre ese formato.
- Paleta base: --bg-void #0a0a0f / --accent-aether #7b2fff / --accent-gold #f0a500 / --accent-blood #c0392b
